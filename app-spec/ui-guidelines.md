# UI Guidelines

## Palette
- Noir : #0B0B0C  
- Beige : #EAE3D9  
- Doré : #C5A572  
- Bleu foncé : #0E2A47  
- Gris : #6B7280  

## Style
Minimal, zen, arrondi 8px, ombres douces, espace généreux.

## Typographies
- Inter ou Outfit  
- Titres bold, texte régulier  

## Composants
- Cards, Buttons, Tabs, Modal, Toast  
- Icônes fines (Lucide, Phosphor)
